ngx-twitter

ngx-twitter is an Angular component library that makes it easy to embed tweets in your application, using the https://platform.twitter.com/widgets.js script. When the module is loaded, the script is loaded and the tweets are displayed after that. You can pass use a single ngx-tweet component and pass it an tweet string ID, or you can use the ngx-tweet-list component which allows you to pass in a list of tweets, and embeds each of those.

You can also pass in an options object: 

public cards: string; 		// defaults to 'hidden'
public conversation: string; 	// defaults to 'none'
public theme: string; 		// defaults to 'light'
public linkColor: string; 	// defaults to '#55acee'
public align: string; 		// defaults to 'center'
public lang: string; 		// defaults to 'en'
public width: number; 		// defaults to 325
public dnt: boolean; 		// defaults to true
